# PMAF Dashboard

Pharmaceutical Market Attractiveness Framework (PMAF) dashboard developed for **Trilytics 2026 (Sun Pharma)**.

## Features

- Executive Overview
- District Explorer
- District Comparison
- Rankings
- Analytics Dashboard
- PMAF Methodology

## Tech Stack

- Streamlit
- Plotly
- Pandas
- NumPy

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Dataset

The dashboard uses the processed district-level PMAF dataset located at:

```
data/final/district_rankings.csv
```
